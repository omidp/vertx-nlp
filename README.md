# vertx-nlp
practice with vertx, tika, lucene for a simple Natural Language Processing example

## how to build

```
mvn clean install
```

## how to run

* run NlpVerticle
* open browser type

```
http://127.0.0.1:8585/parse?url=http://jedlab.ir/home.seam
```

then 

```
http://127.0.0.1:8585/find/جاوا,جدلب,اخبار
```
